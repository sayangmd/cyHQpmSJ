# 项目简介

## 核心功能
- **管理员功能**:
  - 个人中心
  - 问卷管理
  - 调查管理
  - 题目管理
  - 用户管理
  - 新闻资讯管理
  - 基础数据管理
  - 轮播图管理
- **用户功能**:
  - 问卷列表
  - 填写问卷
  - 新闻资讯
  - 问卷调查记录
  - 个人中心
  - 注册登录
- **工具**:
  - Idea/Eclipse
  - Vscode
  - Navicat

## 适用场景
- 适用于需要进行在线问卷调查的机构或个人。
- 适合收集市场调研、学术研究、用户反馈等数据。

## 技术特点
- 采用Spring Boot框架，保证系统的高效和稳定性。
- 使用Vue.js进行前端开发，提供良好的用户体验。
- 支持JDK 1.8、Node 16、MySQL 8等主流技术。
- 使用Maven 3.6进行项目管理和构建。
